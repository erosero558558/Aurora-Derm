# AG-110 - Turnero sala N: privacidad operativa kiosco+TV (auto-reset sesion + control campanilla)

## Objetivo

Endurecer operacion de sala para proteger privacidad en kiosco y controlar audio de campanilla en TV sin depender de cambios backend.

## Cambios implementados

- `src/apps/queue-kiosk/index.js`
    - Nuevo guard de sesion por inactividad con countdown de privacidad.
    - Auto-reset de sesion (formularios + ticket + conversacion del asistente) tras timeout configurable.
    - Boton operativo `Nueva persona / limpiar pantalla` y atajo `Alt+Shift+L`.
    - Mantiene polling de cola y manual refresh previos, ahora con limpieza de pantalla operativa.
- `src/apps/queue-display/index.js`
    - Control persistente de campanilla (`Campanilla: On/Off`) con almacenamiento local.
    - Atajo `Alt+Shift+M` para alternar mute en sala.
    - `playBell()` respeta estado mute para evitar ruido no deseado.
- Bundles regenerados:
    - `js/queue-kiosk.js`
    - `js/queue-display.js`
- QA (Playwright):
    - `tests/queue-kiosk.spec.js`
        - nuevo escenario de reset manual + reset por inactividad.
    - `tests/queue-display.spec.js`
        - nuevo escenario de mute persistente y toggle por teclado.

## Validacion ejecutada

- `npx eslint src/apps/queue-kiosk/index.js src/apps/queue-display/index.js tests/queue-kiosk.spec.js tests/queue-display.spec.js` (ok)
- `npm run build` (ok)
- `npx playwright test tests/queue-kiosk.spec.js tests/queue-display.spec.js --reporter=line` (ok, 6 passed)
