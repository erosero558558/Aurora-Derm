# AG-011 - Performance split and lazy-init hardening

## Scope
- Reduce initial `script.js` payload.
- Move non-critical runtime modules to dynamic imports.
- Split engagement bundle so forms and reviews load independently.

## Changes applied
- Updated `js/main.js` to lazy-load non-critical runtimes:
  - `engagement`
  - `gallery`
  - `ui`
  - `reschedule`
  - `success-modal`
- Refactored static imports to dynamic wrappers:
  - `js/router.js`
  - `js/booking.js`
  - `js/i18n.js`
  - `js/analytics.js`
- Split engagement bundle outputs:
  - Updated `src/bundles/engagement.js` (reviews only)
  - Added `src/bundles/engagement-forms.js` (forms only)
  - Updated `rollup.config.mjs` with `js/engines/engagement-forms-bundle.js`
  - Updated `js/engagement.js` to load reviews/forms bundles independently
