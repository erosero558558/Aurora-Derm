# AG-233 - Wire admin copiloto keyboard shortcut

## Resumen

- El admin v3 ahora cablea `focusAgentPrompt` dentro de `attachKeyboardShortcuts`, para que el copiloto pueda abrirse desde los atajos globales.
- El handler global separa `Ctrl+K` para la barra de comandos y agrega `Ctrl+Shift+K` como alias directo del copiloto, manteniendo `Alt+Shift+I`.
- La ayuda visible del shell y el contrato DOM se actualizaron para reflejar el acceso al copiloto y el bundle de `admin.js` se regenero con el chunk vigente.
- El spec de navegacion administra capabilities `adminAgent` en el mock de auth para validar la apertura real del panel en runtime.
