# AG-225 - Retire legacy public chat booking specs against public V6 surfaces

## Summary

- Replaced the legacy chat-widget booking assumptions in `tests/chat-booking-calendar-errors.spec.js` with the current `public-v6` fallback contract for root redirect, booking-status messaging, telemedicine handoff, and current assistance surfaces.
- Replaced the old chat-widget viewport assertions in `tests/chat-mobile-layout.spec.js` with responsive layout checks for the `public-v6` drawer and search overlay.
- Captured the current responsive behavior where search is header-driven on desktop and drawer-driven on narrow and mobile layouts.
