# AG-097 - Turnero sala A: backend queue + printer

Fecha: 2026-02-26
Owner: Ernesto
Executor: codex

## Cambios

- `controllers/QueueController.php`
    - Nuevo controlador API para turnero: `queue-state`, `queue-checkin`, `queue-ticket`, `queue-call-next`, `queue-ticket (PATCH)`, `queue-reprint`.
    - Mutaciones con `with_store_lock` + `write_store` atomico.
    - Respuestas con estado de impresion (`printed`, `print.errorCode`, `print.message`).
- `lib/QueueService.php`
    - Lógica de negocio del MVP:
        - cola unica para 2 consultorios,
        - check-in de cita (telefono + hora + fecha),
        - no duplicacion de ticket por cita,
        - prioridad fija `appt_overdue > appt_current > walk_in`,
        - llamado manual por consultorio,
        - acciones de ticket (`re-llamar`, `completar`, `no_show`, `cancelar`, `reasignar`),
        - resumen operativo para admin.
- `lib/TicketPrinter.php`
    - Integracion ESC/POS TCP para impresora de red (`host:port`), con fallback no bloqueante.
    - Variables soportadas:
        - `PIELARMONIA_TICKET_PRINTER_ENABLED`
        - `PIELARMONIA_TICKET_PRINTER_HOST`
        - `PIELARMONIA_TICKET_PRINTER_PORT`
        - `PIELARMONIA_TICKET_PRINTER_TIMEOUT_MS`
- `lib/db.php`
    - Nueva tabla `queue_tickets` + indices (`status`, `createdAt`, `dailySeq`).
- `lib/storage.php`
    - `queue_tickets` agregado en payload default, normalizacion, migracion JSON->SQLite, lectura y escritura transaccional.
- `lib/models.php`
    - Nuevo normalizador `normalize_queue_ticket(...)`.
- `lib/routes.php`
    - Registro de rutas `queue-*`.
- `lib/ApiConfig.php`
    - Rate-limits para `queue-*`.
    - Endpoints publicos para `queue-state`, `queue-checkin`, `queue-ticket`.
- `controllers/AdminDataController.php`
    - Se incluye `queueMeta` en `/api.php?resource=data`.
    - Import admin soporta `queue_tickets`.
- `api.php`
    - Registro de `QueueController`.
- `tests/test_queue_service.php`
    - Test backend para:
        - creacion walk-in,
        - check-in sin duplicacion,
        - prioridad de llamado,
        - acciones de estado,
        - fallback de impresora deshabilitada.
