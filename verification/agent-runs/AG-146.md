# AG-146 - Safe cron auto-sync for public deploy

Date: 2026-03-02
Owner: Ernesto
Executor: codex

## Goal

Add a server-side cron wrapper that auto-publishes production only when `origin/main` changes, without overlapping runs or overwriting a dirty working tree.

## Changes

- Added `bin/deploy-public-v3-cron-sync.sh`
- Added `docs/PUBLIC_V3_CRON_INSTALL.md`

## Safety properties

- non-blocking lock with `flock`
- exits cleanly when no remote changes exist
- aborts if server repo is dirty
- reuses the existing public deploy script
- logs to `/var/log/pielarmonia-public-deploy.log`

## Validation

- `bash -n bin/deploy-public-v3-cron-sync.sh`

## Operator command

```bash
cd /var/www/figo
chmod +x bin/deploy-public-v3-cron-sync.sh
crontab -l 2>/dev/null | { cat; echo '* * * * * REPO=/var/www/figo LOG_PATH=/var/log/pielarmonia-public-deploy.log /usr/bin/env bash /var/www/figo/bin/deploy-public-v3-cron-sync.sh'; } | crontab -
```
