# AG-238 - Mantener útil la acción clínica del dashboard con fallback operativo

## Resumen

- añadí la tarjeta "Frente clinico" al overview del dashboard para que el panel muestre un punto operativo claro aunque no existan sesiones clínicas abiertas todavía
- extendí `buildClinicalHistoryActions()` para aceptar el estado completo del dashboard y ofrecer una acción de fallback basada en telemedicina, patient flow y readiness clínica
- conecté el render del dashboard a ese contexto ampliado y cubrí el comportamiento con specs del panel admin
