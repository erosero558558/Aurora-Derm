# AG-106 � Turnero sala J: atajos operativos y bulk workflow en admin queue

## Objetivo

Mejorar operacion del turnero en admin con atajos de teclado y acciones masivas sobre la vista filtrada, manteniendo QA del modulo.

## Cambios implementados

- `src/apps/admin/modules/queue.js`
    - Nuevas acciones masivas: `completar`, `no_show`, `cancelar` sobre tickets visibles y elegibles.
    - Botonera bulk dentro de toolbar de triage con conteo dinamico por accion.
    - Estado `bulkActionInFlight` para bloquear doble ejecucion y mostrar progreso.
    - API interna reutilizable para acciones de ticket (`requestQueueTicketAction`) y opcion `silent/skipRender`.
    - Nuevas utilidades exportadas: `runQueueBulkAction`, `setQueueFilter`, `focusQueueSearch`.
    - Hint visual de atajos de teclado en toolbar.
- `src/apps/admin/index.js`
    - Atajos en seccion Turnero:
        - `Alt+Shift+J` llamar C1
        - `Alt+Shift+K` llamar C2
        - `Alt+Shift+U` refrescar cola
        - `Alt+Shift+F` foco en busqueda
        - `Alt+Shift+L` filtro SLA
        - `Alt+Shift+O` limpiar filtro (all)
        - `Alt+Shift+G/H/B` bulk completar/no_show/cancelar
    - Quick-command extendido para turnero (`turnero sla`, `bulk completar/no show/cancelar`, etc.).
- `tests/admin-queue.spec.js`
    - Nuevo escenario E2E: atajos de teclado + filtro SLA + accion masiva en cola.

## Validacion ejecutada

- `npx eslint src/apps/admin/index.js src/apps/admin/modules/queue.js tests/admin-queue.spec.js` ?
- `npm run build` ?
