# AG-126 — Turnero sala QA: atajos numpad seguros (atajo 0 + Enter principal)

## Resumen

Se cerró un bloque de QA para endurecer atajos del turnero en admin y documentar mejor la operación de recepción.

## Cambios

- `tests/admin-queue.spec.js`
    - se reforzó la prueba de ayuda rápida para validar apertura/cierre con atajo `Alt+Shift+0` y continuidad de `Numpad +` (re-llamar).
    - se agregó guardrail explícito en la prueba cross-platform:
        - `Enter` principal (`location=1`) no llama turnos.
        - `Enter` numpad (`location=3`) sí llama.
    - se estabilizó la prueba de decimal regional esperando actualización visual de `C1` antes del segundo `NumpadDecimal`.
- `docs/RUNBOOK_TURNERO_SALA.md`
    - se añadió guía operativa de “Modo 1 tecla” (opcional) en flujo de recepción.
    - se mantiene como default el flujo seguro de 2 pasos para minimizar errores.
