# AG-131 — Turnero sala XIII: hardening anti-flake de suite QA queue

## Objetivo

Eliminar flakiness intermitente en la suite Playwright del turnero (admin + kiosco + TV) bajo ejecucion paralela.

## Cambios implementados

1. `playwright.config.js`

- Se agrego `use.serviceWorkers = 'block'` para evitar interferencia de cache/SW en mocks de API.

2. `tests/admin-queue.spec.js`

- En el test `triage de cola aplica filtros SLA y busqueda por ticket`:
    - se agrego mock explicito de `resource=queue-state` con `queueState` de prueba.
    - se aumento timeout de `#adminDashboard` a `15000ms` para robustez bajo carga paralela.

3. `tests/queue-display.spec.js`

- En el test `permite limpiar snapshot local desde controles de contingencia`:
    - se aumento timeout de espera para `#displayConsultorio2` a `10000ms`.

## Validacion ejecutada

- `npx playwright test tests/admin-queue.spec.js tests/queue-kiosk.spec.js tests/queue-display.spec.js tests/queue-integrated-flow.spec.js` ✅
