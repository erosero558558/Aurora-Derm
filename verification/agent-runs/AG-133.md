# AG-133 - Turnero sala XV: TV estrella de sala (claridad, privacidad y audio confiable)

Fecha de cierre: 2026-02-28  
Owner: Ernesto  
Executor: codex

## Objetivo

Mejorar la experiencia de TV de sala para lectura a distancia y operacion diaria: llamado mas claro por consultorio, campanilla mas confiable y fallback visual cuando audio no dispara.

## Cambios implementados

1. `src/apps/queue-display/index.js`

- Se reforzo el flujo de campanilla con:
    - flash visual de llamado (`display-bell-flash`) para no depender solo de audio.
    - cooldown de campanilla para evitar ruido repetido por rafagas.
    - priming de audio por gesto de usuario (`pointerdown`/`keydown`) y evento de diagnostico.
    - hint operativo cuando el navegador bloquea autoplay de audio.
- Se agrego boton de operacion `Probar campanilla` (`#displayBellTestBtn`).
- Se agrego atajo de teclado `Alt+Shift+B` para prueba de campanilla.
- Se ajusto el mensaje principal de llamado para mayor claridad:
    - antes: `ticket + iniciales + consultorio`
    - ahora: `Consultorio X · Turno A-000` + linea de apoyo con iniciales.
- Se mantuvo privacidad estricta (solo ticket + iniciales).

2. `sala-turnos.html`

- Se actualizo copy de cabecera a "lectura grande".
- Se agrego hint visible de atajos operativos para staff:
    - `Alt+Shift+R` refrescar
    - `Alt+Shift+M` campanilla on/off
    - `Alt+Shift+B` prueba de campanilla

3. Build

- Se regenero `js/queue-display.js` via build del proyecto.

## Resultado operativo

- TV de sala comunica llamado actual con mejor jerarquia visual.
- Si el audio del navegador/TV esta restringido, el sistema sigue marcando llamados con señal visual y mensaje operativo claro.
- Recepcion tiene prueba manual y atajo de validacion rapida de campanilla.
