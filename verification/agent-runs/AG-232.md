# AG-232 - Consolidar primer cluster admin queue api mocks

- Fecha: 2026-03-14
- Owner: Ernesto
- Executor: codex

## Resumen

Se consolidó el primer cluster repetido de mocks API en
`tests/admin-queue.spec.js` usando un wrapper local `installAdminQueueApiMocks`
montado sobre el helper compartido `installBasicAdminApiMocks`.

El cambio cubre los cuatro primeros casos del spec:

- `cola muestra guia accionable para apoyos activos y enfoca el ticket correcto`
- `guia de recepcion ofrece atajos contextuales por motivo y los ejecuta`
- `agenda lee disponibilidad real y abre el dia pedido para conflicto horario`
- `permite llamar siguiente ticket en consultorio 1`

Con esto se eliminó el baseline repetido de `features`, `/data`,
`queue-state`, `health` y `funnel-metrics`, dejando solo las ramas especiales
por test.
