# AG-125 — Turnero sala estrella: kiosco premium con bienvenida guiada y ayuda persistente

## Resumen

Se reforzó la experiencia del kiosco para primera interacción y operación diaria, manteniendo flujo rápido de turnos/check-in con ayudas visibles y atajos claros.

## Cambios

- `kiosco-turnos.html`:
    - nueva barra de acciones rápidas (`Tengo cita`, `No tengo cita`) para entrar al flujo correcto en un toque.
    - panel de ayuda rápida con pasos visibles y cierre directo.
    - estado de progreso en vivo para guiar al usuario durante validación/envío/éxito/error.
- `src/apps/queue-kiosk/index.js`:
    - estado UX para ayuda, flujo activo y secuencia de bienvenida.
    - inyección de estilos "star mode" para resaltar flujo activo y mantener jerarquía visual estable.
    - atajos de teclado de soporte:
        - `Alt+Shift+H` para ayuda,
        - `Alt+Shift+1` para check-in,
        - `Alt+Shift+2` para walk-in.
    - mejoras de mensajes de progreso en submit de check-in/walk-in (éxito, error y modo offline).
- `js/queue-kiosk.js` regenerado por build.

## Archivos
