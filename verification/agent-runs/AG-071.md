# AG-071 — Frontend admin: normalizar copy ES y acentos visibles

## Resumen
Se normalizó copy visible en español (acentos y microcopy) del panel Admin para login, navegación, filtros, toasts y mensajes de disponibilidad, manteniendo comportamiento funcional sin cambios de backend.

## Archivos modificados
- admin.html
- admin.js
- src/apps/admin/index.js
- src/apps/admin/modules/ui.js
- src/apps/admin/modules/availability.js

- Navegación y labels: `Navegación`, `Menú`, `rápida`, `administración`.
- Estados y filtros: `No asistió`, `Duplicar a mañana`, `Errores por código`.
- Toaster/UI: `Éxito`, `Cerrar notificación`, `Portapapeles vacío`, `Día copiado`.
- Login y auth feedback: `Código`, `sesión`, `Contraseña`.
- Servicios médicos: `Acné`, `Detección de Cáncer de Piel`, `Dra. Narváez`.
