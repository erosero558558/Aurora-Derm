# AG-129 — Turnero sala XI: calibracion de tecla externa por estacion (numpad wireless)

## Fecha

- 2026-02-28

## Objetivo

- Permitir que cada estación (C1/C2) capture una tecla externa de llamado cuando el numpad inalámbrico no reporta `NumpadEnter` estándar.

## Cambios aplicados

- `src/apps/admin/index.js`
    - Se agregó persistencia local de tecla externa (`queueCallKeyBindingV1`).
    - Se incorporó flujo de calibración desde el panel de estación:
        - botón dinámico `Calibrar tecla externa`.
        - captura de la siguiente tecla presionada (con `Esc` para cancelar).
        - botón `Quitar tecla externa` cuando existe binding activo.
    - Se añadió detección de teclado por firma (`code`, `key`, `location`) para permitir llamado con teclas externas no-numpad.
    - Se integró el binding personalizado en el atajo de llamado (equivalente a `Numpad Enter`) sin romper el resto de atajos.
    - Se añadieron guardrails:
        - no captura modificadores,
        - cancelación explícita,
        - desactivación automática de captura al salir de sección `queue` o al logout.
    - Se añadieron eventos operativos `piel:queue-ops` para captura/guardado/limpieza de tecla.
    - Se actualizó hint operativo en `Control de estación` y panel de atajos para mostrar tecla externa activa.
