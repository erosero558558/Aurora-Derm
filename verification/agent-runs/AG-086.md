## Implemented

- Added exported helper `hasAvailabilityDraftChanges()` in availability module.
- Added guard in admin navigation flow:
    - When leaving `availability` with unsaved draft changes, prompts confirmation.
    - If user cancels, navigation is blocked and current section/hash is preserved.
    - If user accepts, navigation continues normally.
- Added Playwright coverage for compact sidebar flow:
    - Dismiss prompt keeps user in `#availability`.
    - Accept prompt allows transition to `#appointments` and closes overlay.
