# Public V6 Copy System (R7)

Version: 2026-03-14-r7
Scope: public-v6 ES canonico (navigation, home, hub, services, telemedicine, legal, software, shell labels)

## Objetivo editorial

Hacer que la web en espanol suene humana, clara y cercana para Ecuador:
menos brochure, menos solemnidad, menos frases que parecen hechas para
impresionar, y mas lenguaje que ayude a entender que pasa, que sigue y por
que conviene una ruta u otra.

## Modelo de voz ES

- Registro: profesional en `usted`.
- Tono: cercano, humano y sereno, sin rigidez institucional.
- Ritmo: frases claras, con aire conversacional, sin exceso de adornos.
- Orden mental: `que le pasa -> que ayuda -> que sigue`.
- Centro del lenguaje: beneficio real, decision clara y siguiente paso util.
- Ecuacion editorial: explicar bien pesa mas que sonar premium.

## Reglas anti-IA

- No usar frases que suenen a manifiesto, brochure o pitch generico.
- No repetir la misma promesa con palabras distintas en cada bloque.
- No armar triadas vacias del tipo `claridad, criterio y seguimiento` si no
  aterrizan una accion concreta.
- No escribir como si cada seccion quisiera sonar "elegante".
- No usar lenguaje corporativo para rutas de pacientes.
- No meter tecnicismos internos visibles al usuario.
- No esconder la accion real detras de un texto bonito.

## Guardrails clinicos y legales

- No prometer resultados absolutos.
- No usar `garantizado`, `100%`, `cura definitiva` ni equivalentes.
- No sugerir ausencia de riesgo.
- No reinterpretar politicas legales mas alla de lo que el sitio soporta.
- Legal debe seguir siendo claro, pero no coloquial.

## Restricciones de longitud

- Hero category: 1-3 palabras.
- Hero title: 6-12 palabras.
- Hero description: 12-28 palabras.
- News strip headline: 10-22 palabras.
- Card title: 4-12 palabras.
- Card copy: 10-24 palabras.
- Hub intro title: 6-16 palabras.
- Hub intro deck: 10-26 palabras.
- Tele lead: 8-22 palabras.
- Booking status title: 3-8 palabras.
- Booking status description: 12-30 palabras.
- Service lead: 12-24 palabras.
- Service FAQ answer: 12-34 palabras.
