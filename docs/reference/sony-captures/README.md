## Nota de evidencia

## Uso en auditoria

La auditoria V6 (`bin/audit-public-v6-visual-contract.js`) mapea cada checkpoint a un bloque inspirado en estas capturas para verificar:

1. estructura DOM,
2. jerarquia tipografica,
3. proporciones de layout,
4. comportamiento interactivo.
